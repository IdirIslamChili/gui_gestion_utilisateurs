# _Gestion des utilisateurs via une interface graphique (GUI)_
## **Auteur :** Idir Islam Chili

### Description :
Ce projet est une application de gestion des utilisateurs développée en Python avec une interface graphique (GUI). 
Elle permet d’effectuer des opérations de base sur une base de données MySQL afin de gérer efficacement des comptes utilisateurs.
Les données seront enregistrées dans une base de données MySQL et l’application sera structurée selon une architecture en couches, afin de séparer clairement les responsabilités (données, logique, métier, interface graphique.)

### ⚙️ Technologies utilisées :
Python 🐍,
customTkinter ,
MySQL.
